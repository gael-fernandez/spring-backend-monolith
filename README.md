# Spring Backend Monolith

Monolithic backend built with Java 17 and Spring Boot to learn core backend fundamentals and clean architecture.

## Tech Stack
- Java 17
- Spring Boot
- Maven
- Embedded Tomcat

## Project Structure
- **Controller layer**: Handles HTTP requests and responses.
- **Service layer**: Contains business logic and decision making.
- **Configuration**: Application properties and Spring Boot setup.

## Current Features
- Application bootstrap with Spring Boot
- Health check endpoint (work in progress)

## Goals
- Understand backend fundamentals (JVM, threads, memory)
- Apply clean separation of responsibilities
- Practice dependency injection and stateless design
- Build a realistic monolithic backend as a learning project

```bash
./mvnw spring-boot:run
